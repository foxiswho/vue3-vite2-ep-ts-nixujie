# 须弥戒后台
elementPlus TypeScript vue3 Vite 2.0 Vue-Router 4.0 Echart 5.0 Axios

# 安装 yarn
```bash
npm install yarn -g
```
# 初始化
```bash
yarn 
```
# 开发运行
```bash
yarn dev
```

# 编译
```bash
yarn build
```
编译后目录位于根目录下 `dist`
